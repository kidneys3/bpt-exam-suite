# BPT Exam Suite

## Setup
1. Go to repo Settings → Pages → Source: **GitHub Actions**
2. Push any change — the workflow auto-deploys in ~30 seconds

## Updating the app
Replace `index.html` with the latest version and push.
The site rebuilds automatically — no manual steps needed.
